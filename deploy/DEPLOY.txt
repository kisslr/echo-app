=== 回声 - 部署指南 ===

═══════════════════════════════════
推荐方案: Sealos（国内，云端AI可用）
═══════════════════════════════════
平台: sealos.run
费用: 实名送5元（≈1000次AI对话）
优点: 国内服务器 → 能直连 vivo 蓝心平台 API → 大模型真正可用
域名: https://xxx.cloud.sealos.run（永久有效，HTTPS）

详细步骤见 deploy/SEALOS.md

═══════════════════════════════════
当前方案: HuggingFace Spaces（海外）
═══════════════════════════════════
平台: huggingface.co/spaces
费用: 永久免费
缺点: 海外服务器 → 无法访问 vivo 国内 API → 云端AI走本地引擎兜底
域名: https://break66-echo-app.hf.space

上传步骤:
1. 打开 https://huggingface.co/spaces/break66/echo-app
2. Files 标签 → Add file → Upload files
3. 把项目文件全部拖上去 → Commit
4. App 标签等构建完成（2-5分钟）
5. 访问链接验证

═══════════════════════════════════
本地运行（评委下载代码包后）
═══════════════════════════════════
1. 解压 echo-app-submission.zip
2. 双击 install.bat 安装依赖
3. 双击 start.bat 启动服务
4. 手机连同一WiFi，浏览器访问 http://电脑IP:7860

本地运行时，如已配置 BLUELM_API_KEY 且网络可访问 vivo API，云端AI可用。
未配置时会自动使用本地自然语言引擎兜底。
一键测试: python test_api.py
