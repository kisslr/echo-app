# 回声 · 国内部署方案（Sealos）

## 为什么选 Sealos
- 国内服务器，能直连 `api-ai.vivo.com.cn`（vivo 蓝心平台）
- 支持 Docker 部署，直接用现有 Dockerfile
- 免费额度够演示（5元注册金 + 按量计费，闲置不扣费）
- 提供公网 HTTPS 域名，永久有效
- 不休眠，无 API 限制

## 部署步骤

### 第 1 步：注册
1. 打开 `https://sealos.run`
2. 点「立即开始」→ 用手机号注册
3. 实名认证（微信扫码，1分钟）
4. 新用户自动获得 5 元余额

### 第 2 步：部署应用
1. 进入控制台 → 点「应用管理」
2. 点「新建应用」→ 选「容器应用」
3. 镜像来源选「上传代码」→ 上传整个 `echo-app/` 目录
4. Sealos 自动检测 Dockerfile 并构建
5. 端口设置：容器端口填 `7860`
6. 内存分配：256MB（够用）
7. 点「部署」

### 第 3 步：获取链接
1. 部署完成后 → 点「访问地址」
2. 系统自动分配域名：`https://xxx.cloud.sealos.run`
3. 复制这个链接 → 填入 PPT、README、提交系统

### 第 4 步：设置环境变量（可选）
在 Sealos 应用设置 → 环境变量中添加：
```
BLUELM_API_KEY=你的vivo比赛AppKey
```
不设置也可以运行完整产品流程，但云端大模型会降级为本地自然语言引擎。
